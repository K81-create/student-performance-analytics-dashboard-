# Student Performance Analytics Dashboard

A modern web-based dashboard to analyze student academic performance using marks and attendance data.  
It provides key insights like average marks, average attendance, at-risk student detection, and interactive charts for better decision-making.

---

## Features

-  **Semester Filter** – View performance semester-wise
-  **Subject Filter** – Compare subject performance easily
-  **Student Filter** – Track individual student progress
-  **KPI Cards**
  - Average Marks
  - Average Attendance
  - At-Risk Students
  - Total Students
-  **Semester-wise Performance Chart**
-  **Subject-wise Comparison Chart**
-  **Attendance vs Marks Correlation (Scatter Plot)**
-  Fully Responsive UI (Works on laptop)

---

##  Tech Stack

- **Frontend:** React + Vite
- **Language:** TypeScript / JavaScript
- **UI:** Tailwind CSS
- **Charts:** Chart.js / Recharts (whichever you used)
- **Icons:** Lucide / React Icons
- **Deployment:** GitHub Pages / Vercel / Netlify
